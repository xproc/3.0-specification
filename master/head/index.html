2 XProc 3.0

XProc 3.0 is defined normatively by two specifications:

  • XProc 3.0: A Pipeline Language defines the semantics of XProc pipelines.

  • XProc 3.0: Standard Step Library defines the required atomic steps. All conformant implementations implement these steps. General features of atomic steps are described in XProc 3.0 Steps: An Introduction.
